# Sample 3MF Files

## inputs/

Source models for testing the post-processing tools.

| File | Description |
|------|-------------|
| `Cube.3mf` | Simple 20mm cube — unsliced model for layer pattern assignment |
| `Cube.gcode.3mf` | Same cube, sliced with Bambu Studio (contains G-code) — required for interlocking perimeters, topology infill, and wall-infill interlock |

## outputs/

Example results generated by the post-processing tools.

| File | Generated by | Description |
|------|-------------|-------------|
| `Cube_F1-1_F2-2_F1-3.3mf` | `layer_pattern` | Layer pattern F1:1, F2:2, F1:3 (repeating) |
| `Cube_interlocking.3mf` | `interlocking_perimeters` | Wall 2 offset by half layer height |
| `cube_interlock.3mf` | `interlocking_perimeters` | Alternate interlocking config |
| `cube_wall_interlock.3mf` | `wall_infill_interlock` | Sinusoidal teeth on walls and infill |

## Usage

```bash
# Layer pattern: assign alternating filaments
python -m multimaterial_3d.postprocessors.layer_pattern \
    samples/inputs/Cube.3mf samples/outputs/my_pattern.3mf \
    --pattern F1:3,F2:2 --layer-height 0.2 --total-height 20

# Interlocking perimeters (requires sliced input)
python -m multimaterial_3d.postprocessors.interlocking_perimeters \
    samples/inputs/Cube.gcode.3mf samples/outputs/my_interlock.3mf

# Topology-optimized infill (requires sliced input)
python -m multimaterial_3d.postprocessors.topology_infill \
    samples/inputs/Cube.gcode.3mf samples/outputs/my_topology.3mf

# Wall-infill interlocking (requires sliced input)
python -m multimaterial_3d.postprocessors.wall_infill_interlock \
    samples/inputs/Cube.gcode.3mf samples/outputs/my_wall_interlock.3mf
```
